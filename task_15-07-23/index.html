Git Commands:

  1. Git --Version
    • Check your git is properly installed or not by using and also check your git version


  2. Git config --list
    • Check your user config in Git Bash by using command


  3. git config --global user.name “username”
    git config --global user.email “useremail”
    • If there are no user then add by using command


  4. git config --unset --global user.name "username"
    git config --unset --global user.email "useremail"
    • If you want to remove any user from your git then use command


  5. Now go to GitHub, click on your profile, choose your repository menu then click on New to create a repository




  6. git clone "repository link"
    • Once repository create then copy given URL and past it in your code editor like VS Code or in git bash by using following command


  7. Open this repository in VS Code or any other code editor software
    • In VS Code install an extension to connect your GitHub Repository

    • Click on Account icon which are showing at left bottom corner and sign in with your GitHub account

    • After completion of these task create file like index.html and write your code. Save it


  8. git add . && git add "file-name"
    • Open terminal in vs code and use command to stage your code
      (git add . ) for staging all code
      (git add "file-name") for staging particular file


  9. git commit -m "commit message"
    • After staging, save your code in local repo by using following command


  10. git push origin "branch-name"
    • After commit, save it on your central repo (GitHub) by using


  11. git remote add origin "URL of repository"
    • In above a, origin is remote access name if there is any right access permission error are showing then use following command for remote access


  12. Now check repository, your find your code successfully store on GitHub



  13. git status
    • After complete above step, use these command to check status of your workspace , see commit, difference etc. using following command. *_to check workspace status_*


  14. git log
    • to show the list of commit history


  15. git log --oneline
    • to show the list of commit history in one line


  16. git show "commit-id"
    • to show the details of particular commit
